### 对代码工作原理的理解

过程：Java编译器将源代码编译成字节码；SteganographyEncoder类负责将字节码编码进所选择的图片中。运行时，通过对应的Decoder类进行解码，loadClass方法通过向上委托向下查找的原则进行搜索，所以只有在没有搜到同名类的情况下才会调用SteganographyClassLoader来解码，将其解码成类对象运行。
